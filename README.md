# /images/foods

Every dish image here (`paneer-tikka.svg`, `butter-chicken.svg`, etc.) is a
locally generated, color-coded SVG graphic — no internet connection needed,
so images always load, including on Live Server, offline, or GitHub Pages.

To use your own photography instead:

1. Drop your images here, e.g. `paneer-tikka.jpg`, `butter-chicken.jpg`.
2. In `index.html` / `menu.html`, replace the matching
   `<img src="images/foods/paneer-tikka.svg">` with
   `<img src="images/foods/paneer-tikka.jpg">`.
3. Keep images under ~300KB and roughly 4:3 ratio for consistent card sizing.
